April 24 is JS Naked Day!

Join us for the yearly festivities on April 24 of going without JavaScript on the Web!

The idea behind JS Naked Day is to promote the rule of least power. That is use HTML for semantic markup, CSS for styling and JavaScript for interactivity. Plain and simple your website should work without JavaScript enabled.

Why April 24?

Well in some countries that date would be written as 424 which is also the HTTP error code for failed dependency. JS Naked Day is to promote the fact that websites should work without a JavaScript dependency.

How to participate

Website owners

On April 24, simply remove all JavaScript from your website. JS Naked Day lasts for one international day. Technically speaking, it will be April 24 somewhere in the world for 50 hours, from 00:00, April 24 UTC+14:00 to 00:00, April 24 UTC-12:00 (10:00, April 23 UTC+00:00 to 12:00, April 25 UTC+00:00). This is to ensure that everyone’s website will be publicly nude for the entire world to see at any given time during April 24.

You can link to this page to let your visitors know about the nudity of your website! This option is for those who feel a need to give their visitors a reference as to what’s going on. This is not about getting traffic or making money. There are no ads on this site, nor will there ever be. This is about you, the people; getting naked.

You can instead add your website(s) here by creating an issue or updating the proper file! Note: Alphabetical order should prevent merge conflicts.

Website surfers

On April 24, disable JavaScript in your browser (Chrome, Edge, Firefox, Safari) and surf the web. For sites that do not work without JavaScript consider reporting a bug.
